Library Management System (Java)
Overview

This is a desktop-based Library Management System developed using Java Swing. It allows librarians and members to manage books, users, and transactions through a graphical interface.

Features
Add, remove, and search books
Register and manage members
Add librarians
Issue and return books
View available books
File-based data storage
Technologies Used
Java
Swing (GUI)
File Handling (TXT files)
How to Run
Open in any Java IDE (IntelliJ, Eclipse, etc.)
Compile LibraryMgt.java
Run the program
Note

Make sure all .txt files are in the same folder.
